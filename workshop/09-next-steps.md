## 🔑 Three Things to Remember

### 1. CLI = Speed + Repeatability + Precision

Run the same email triage every Monday. Get the same structured output. Every time. Fast.

### 2. Match the Tool to the Task

| Situation | Best Tool |
|-----------|-----------|
| Repeating this task regularly | **Copilot CLI** |
| Need structured output fast | **Copilot CLI** |
| Triaging lots of information | **Copilot CLI** |
| Exploring something new | **GitHub Copilot Chat** |
| Brainstorming with back-and-forth | **GitHub Copilot Chat** |
| Working with files side by side | **VS Code + Copilot** |
| Editing and iterating on output | **VS Code + Copilot** |

> **Pro tip:** Start in GitHub Copilot Chat to figure out what you need. Move to VS Code to build and refine. Use CLI to run it fast every time.

### 3. You Don't Need to Memorize Anything

With Copilot CLI you just describe what you want in plain English. The commands and syntax are handled for you.
