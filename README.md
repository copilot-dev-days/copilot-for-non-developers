# 🤖 GitHub Copilot for Non-Developers: Work, Assessment, and more!

A beginner-friendly, hands-on workshop for BPMs, PMs, and anyone who gets work done

Take yourself from zero terminal experience to using GitHub Copilot CLI to triage email, extract meeting decisions, analyze data, and more — all in plain English, right from your terminal.

## 🚀 Quick Start

1. Start at [Part 00: Overview](workshop/00-overview.md) to check your prerequisites
2. Work through each part in order
3. Every demo builds on the previous one — don't skip ahead!

## Prerequisites

- A computer (Windows, Mac, or Linux)
- A GitHub account with [GitHub Copilot](https://github.com/features/copilot) enabled
- A Microsoft 365 account (for the MCP demos in Parts 3–6) — your IT admin may need to grant consent the first time
- No coding experience required

## Who Is This For?

- **BPMs** managing complex cross-team programs
- **PMs** tracking decisions and deliverables
- **Ops & Strategy** teams triaging information
- **Anyone** drowning in email and docs

> If you open Outlook and Teams first every morning — this workshop is for you.
